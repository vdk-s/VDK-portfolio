- 🎨 **Light, professional theme** — HR-friendly design with Indigo/Purple accent

- 🔗 **Project GitHub Links** — Each project card links to its exact GitHub repository

## 🌍 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/vdk-s/portfolio.git
   cd portfolio
   ```

2. Open `index.html` in your browser — **no build step needed!**
   ```bash
   # On Windows
   start index.html

   # Or just double-click index.html in File Explorer
   ```

3. *(Optional)* Use VS Code + Live Server extension for auto-refresh.

## 🚀 Deploy to Vercel

### Method 1 — GitHub Integration (Recommended)
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Select your repository → Click **Deploy**
4. Every push to `main` will auto-deploy ✅

### Method 2 — Drag & Drop
1. Go to [vercel.com/new](https://vercel.com/new)
2. Drag & drop your `portfolio` folder
3. Get a live URL instantly ✅
